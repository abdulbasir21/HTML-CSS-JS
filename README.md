# HTML-CSS-JS 🌐

This repository contains a collection of **front-end mini projects** built using pure **HTML, CSS, and JavaScript** — no frameworks or libraries. Each project focuses on solving a small, practical problem while reinforcing core web development fundamentals.

---

## 📂 Projects Included

### 1. [Secure-Password-Maker-main](./Secure-Password-Maker-main)
A password generator tool that creates strong, secure, and customizable passwords instantly.

### 2. [qr-code-generator-main](./qr-code-generator-main)
A simple tool that generates a QR code based on user input (text/link), which can be downloaded or shared.

---

## 🛠️ Tech Stack

- **HTML5** — Structure and markup
- **CSS3** — Styling and layout
- **JavaScript (Vanilla)** — Logic and interactivity

## 🚀 Getting Started

Each project is a standalone static site — no build tools or dependencies required.

```bash
# Clone the repository
git clone https://github.com/abdulbasir21/HTML-CSS-JS.git

# Navigate into a specific project
cd HTML-CSS-JS/<project-folder-name>

# Open index.html in your browser
```

You can simply open the `index.html` file directly in your browser, or use a live server extension (e.g., VS Code's Live Server) for the best experience.
